# HR Roster Dashboard

An HR Dashboard application built with Flutter to interpret master data into a roster sheet view. It provides pivot-style slicers to filter headcounts and employee details by various dimensions like Factory, Department, Education, and Working Status.

## Features
- **Master Data Interpretation:** View mock employee data processed into a readable roster sheet.
- **Pivot Slicers:** Filter the dataset dynamically by:
  - Factory Location
  - Department
  - Education Level
  - Working Status (Active/Inactive)
- **Real-time Metrics:** As filters are applied, the dashboard dynamically updates to show total headcount, active employees, and inactive/removed employees.
- **Responsive Layout:** Works across Web, Desktop, and Mobile environments with adaptive sidebars and metric cards.

## Getting Started

1. Ensure you have the [Flutter SDK](https://flutter.dev) installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run the app using `flutter run` or start it from your IDE.
